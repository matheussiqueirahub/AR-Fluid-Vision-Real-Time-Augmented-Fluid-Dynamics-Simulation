# AR-Fluid-Vision-Real-Time-Augmented-Fluid-Dynamics-Simulation
Simulador avançado de fluidos em Realidade Aumentada que combina física computacional, visão computacional e interação inteligente. Detecta superfícies, integra partículas em tempo real e permite interação com gestos e objetos, criando experiências imersivas voltadas para pesquisa e aplicações práticas.
